# Publicación documental

El workflow `.github/workflows/pages.yml` construye el repositorio con Jekyll y publica `https://alonsomarcosm.github.io/Proyecto_Big_Data_PMD_ASBD/`.

La reproducción completa continúa siendo local mediante `docker-compose/docker-compose.yml`.
